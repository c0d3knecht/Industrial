## Codeknechts Industrial

## 7Days2Die only Server Side Mod



## Installation
Download the "Industrial" folder and copy inside to your 7 Days to Die /Mods folder. If no /Mods folder than just create one and place mods folders in there.

Needs to be on server only

## Usage
Free to install and have fun with it. No modification or using in other mod packs without consent. Do not mess up with the code, it might corrupt your map data.
Some things in the blocks area are done with purpose and for a learning effect.
